Exploratory Notes & Essential Formulas

Welcome to the foundation of Electromagnetic Field Theory. Before we can analyze electric and magnetic fields, we must master the language they are written in: Vectors. Fields exist in 3D space, meaning we must be comfortable moving between Cartesian, Cylindrical, and Spherical coordinate systems seamlessly.

1. Scalars, Vectors, and Position
  • Scalar: A quantity defined entirely by its magnitude (e.g., Time, Mass, Electric Potential, Work).
  • Vector: A quantity defined by both magnitude AND direction (e.g., Force, Velocity, Electric Field Intensity).
  • Unit Vector (aA): A vector with a magnitude of exactly 1. It is used strictly to indicate direction. aA = A / |A|
  • Position Vector (r): A vector drawn from the origin (0,0,0) to a specific point P(x,y,z).
    r = x ax + y ay + z az
  • Distance Vector (RAB): A vector drawn from point A to point B. It is found by subtracting the coordinates of the origin point (A) from the destination point (B).
2. Vector Multiplication

There are two primary ways to multiply vectors in electromagnetics, and they serve completely different physical purposes.

  • Dot Product (A · B): Also known as the scalar product. It yields a SCALAR. It is physically used to find the projection of one vector onto another, or to find the work done by a force. http://googleusercontent.com/image_content/206 A · B = |A||B|cos(θAB) Key Rule: If two non-zero vectors are orthogonal (perpendicular), their dot product is exactly ZERO (since cos 90° = 0).
  • Cross Product (A × B): Also known as the vector product. It yields a new VECTOR that is mutually perpendicular to both A and B. It is used to calculate magnetic force or the area of a parallelogram. http://googleusercontent.com/image_content/217 A × B = |A||B|sin(θAB) an Key Rule: If two non-zero vectors are parallel, their cross product is exactly ZERO (since sin 0° = 0). Direction is found using the Right-Hand Rule.
3. The Three Coordinate Systems

In EEE 303, you cannot solve every problem in x, y, z. For a long wire, we use cylinders. For a point charge, we use spheres. You must know these geometries intimately.

  • Cartesian (x, y, z): The standard grid. The unit vectors (ax, ay, az) are constant and never change direction no matter where you are in space. http://googleusercontent.com/image_content/233
    Differential Volume: dv = dx dy dz
  • Cylindrical (ρ, φ, z): Imagine a tin can.
    ρ (rho): The radial distance from the z-axis (Radius of the cylinder).
    φ (phi): The azimuthal angle measured from the positive x-axis (0 to 360°).
    z: The exact same height variable as in Cartesian.
    Differential Volume: dv = ρ dρ dφ dz
  • Spherical (r, θ, φ): Imagine the Earth.
    r: The absolute distance from the origin to the point.
    θ (theta): The polar angle measured down from the positive z-axis (0 to 180°). Think of this as latitude.
    φ (phi): The azimuthal angle measured from the positive x-axis (0 to 360°). Think of this as longitude.
    Differential Volume: dv = r2 sinθ dr dθ dφ
4. Point Transformations (Finding Coordinates)

Before you can transform a vector, you often need to transform the point at which the vector is being evaluated. This is pure geometry.

  • Cartesian (x, y, z) ↔ Cylindrical (ρ, φ, z)
    Cartesian to Cylindrical: ρ = √(x2 + y2) | φ = tan-1(y / x) | z = z
    Cylindrical to Cartesian: x = ρ cosφ | y = ρ sinφ | z = z
  • Cartesian (x, y, z) ↔ Spherical (r, θ, φ)
    Cartesian to Spherical: r = √(x2 + y2 + z2) | θ = cos-1(z / r) | φ = tan-1(y / x)
    Spherical to Cartesian: x = r sinθ cosφ | y = r sinθ sinφ | z = r cosθ
5. Vector Transformations (The Matrix Method)

Transforming a point is easy, but transforming a Vector Field (like A = 2x ax + y ay) into Cylindrical or Spherical components requires multiplying by a transformation matrix. Why? Because the unit vectors themselves point in different directions!

The Dot Product Hack: To find the ρ component of a Cartesian vector, you simply dot the entire vector with aρ. Because ax · aρ = cosφ and ay · aρ = sinφ, you get the formulas below.

  • Cartesian to Cylindrical Vector Transformation:
    If you are given A = Ax ax + Ay ay + Az az and need A = Aρ aρ + Aφ aφ + Az az:
    Aρ = Ax cosφ + Ay sinφ
    Aφ = -Ax sinφ + Ay cosφ
    Az = Az
  • Cartesian to Spherical Vector Transformation:
    If you are given A = Ax ax + Ay ay + Az az and need A = Ar ar + Aθ aθ + Aφ aφ:
    Ar = Ax sinθ cosφ + Ay sinθ sinφ + Az cosθ
    Aθ = Ax cosθ cosφ + Ay cosθ sinφ - Az sinθ
    Aφ = -Ax sinφ + Ay cosφ
⚠ O'YEMI'S ULTIMATE EXAM HACK:
A classic exam question asks you to evaluate a Cartesian vector field at a specific point, but express the final answer in Cylindrical coordinates.

Example: Given A = y ax + (x+z) ay, evaluate A at point P(-2, 6, 3) in cylindrical coordinates.

Do not panic. Follow these exact steps:
1. Convert the Point: Use x=-2, y=6, z=3 to find ρ and φ. (ρ = √(4+36) = √40. φ = tan-1(6/-2) = 108.43°).
2. Evaluate the Cartesian Vector at P: Plug the x, y, z numbers directly into the given vector A.
Ax = y = 6.
Ay = x+z = -2+3 = 1.
So, at point P, the vector is simply A = 6 ax + 1 ay.
3. Apply the Matrix: Now use the transformation formulas with your calculated Ax, Ay and your angle φ.
Aρ = (6)cos(108.43°) + (1)sin(108.43°)
Aφ = -(6)sin(108.43°) + (1)cos(108.43°)
You are done! You just bypassed pages of painful algebra.
6. Constant Coordinate Surfaces

You must be able to visualize what happens when you hold one variable constant while letting the others vary. This is heavily tested in Multiple Choice.

  • In Cartesian: x = constant forms an infinite plane parallel to the yz-plane.
  • In Cylindrical: ρ = constant forms an infinite cylinder. z = constant forms an infinite plane.
  • In Spherical: r = constant forms a sphere. θ = constant forms a cone. φ = constant forms a semi-infinite plane.
